# MCBE World Prune Functions
A set of functions for pruning a Minecraft bedrock world using [R](https://cran.r-project.org/) and the [rbedrock](https://cran.r-project.org/web/packages/rbedrock/index.html) library by Reed A. Cartwright.
Forked from [functionsPruneWorld.R](https://gist.github.com/Breeze1074/2bc6e71f5f5bad974a425c5a4494504f) by Breeze1074. Last tested on Minecraft 1.26.0.2.0 and rbedrock 0.4.2.9000

## Usage
Check out [rbedrock's GitHub page](https://github.com/reedacartwright/rbedrock) for instructions to install R, RStudio, and rbedrock. You may also need [Rtools](https://cran.r-project.org/bin/windows/Rtools/) to build required packages.

Clone repo using git: `git clone https://github.com/CheeseFort/mcbe-world-prune-functions.git`

[Example usage.R](https://github.com/CheeseFort/mcbe-world-prune-functions/blob/main/Example%20usage.R) and [example world boundaries.xlsx](https://github.com/CheeseFort/mcbe-world-prune-functions/blob/main/example%20world%20boundaries.csv) contain all the information needed to run the scripts.
